Hiperenlaces

Comunmente llamados enlaces, permiten saltar directamente de una página web a otra, o saltar entre puntos de una misma página web (o documento HTML).

Tipos de Hiperenlaces

  • Absolutos: URLs completas. Por ejemplo, la dirección completa del buscador de Google: https://www.google.es/
  • Relativos:
    • Mismo directorio: El nombre del documento HTML que esté en el mismo directorio y su extensión. Por ejemplo: documento.html
    • Subdirectorio: El nombre del subdirectorio, / , y el nombre del documento html y su extensión. Por ejemplo: subdirectorio/documento.html
    • Directorio anterior: ../, seguido del nombre del documento html html y su extensión. Por ejemplo: ../documento.html

Declaración de hiperenlaces

Los hiperenlaces se definen en la etiqueta a, con el atributo href=. El "valor" de href= es la dirección del enlace y el contenido de a será el texto que aparecerá en el hyperenlace. La declaración será así para un enlace absoluto: 

    <a href="https://www.google.es/">GOOGLE</a>

Que se verá así:

GOOGLE

Atributos de los hiperenlaces

Los atributos de a (ademas de href) pueden ser:

  • target: Especifica donde se abrirá el documento vinculado. Puede tener los "valores":
    • "_blank": Abre el documento en una nueva pestaña.
    • "_self": Abre el documento en la misma pestaña donde se hizo click. Es el predeterminado.
    • "_parent": Abre el documento en el marco "padre".
    • "_top": Abre el documento en el cuerpo completo de la ventana.
    • nombre del marco: Abre el documento en el marco identificado con el nombre.
  • download: Atributo booleano (está o no está), no tiene "valor". En lugar de llevarte a otro documento, descarga el contenido de href.

Todos estos atributos se declaran despues de href

Enlaces locales

Esto son saltos dentro del mismo documento HTML. Esto es tan secillo como:

  1. Declarar en cualquier elemento (etiqueta) un atributo name o id
  2. Colocar en el "valor" del href de un hiperenlace el "valor"de:
    • Un atributo id.
    • Un atributo name. Desaprobado en HTML 5.

Tambien se puede hacer un enlace al inicio del mismo documento dando el "valor" "#top" al atributo href.

Se pueden realizar un enlace combinando los parametros locales y globales. De esta manera se creara un enlace a otro documento HTML y, dentro de este, iremos a un punto en concreto. Para declararlo tan solo hay que concatenar una URL, seguida de # y el "valor" de un identificador name o id. Se declara tal y como sigue: 

    <a href="URL#valor_del_identificador">Texto del enlace</a>

Imágenes como hiperenlaces

Se puede utilizar una imagen como hiperenlace anidando la etiqueta img dentro de la etiqueta a, la cual ha de contener su correspondiente href.

Ejemplo de declaración

    <a href="URL#valor_del_identificador" target="_blank"><img src=gifs/tiger_uppercut.gif></img></a>

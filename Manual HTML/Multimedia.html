Etiquetas multimedia

Se utiliza la etiqueta audio para insertar un audio.

Se utiliza la etiqueta video para insertar un video.

Dentro de estas se anidan la etiqueta source dentro de la cual se define la pista de audio.

  • source: Definen los recursos multimedia (básicamente las pistas de video y la pistas de audio).
  • track: Definen los recursos de texto (subtitulos o comentarios en texto).Solo para video

Archivos de audio admitidos

  • .mp3
  • .ogg
  • .wav

Archivos de video admitidos

  • .mp4
  • .webm
  • .ogg

Etiqueta audio

Atributos

  • "autoplay": Autoarranca al cargar la página.Valor booleano (sin valor).
  • "controls": Muestra los controles.Valor booleano (sin valor).
  • "loop": Empieza de nuevo al acabar.Valor booleano (sin valor).
  • "muted": Comienza en silencio (mute).Valor booleano (sin valor).
  • "src": URL donde se ubica el recurso.
    • En el valor de la URL se pude especificar el tiempo de reproducción de los recursos. Para ello solo hay que añadir detrás de la URL #t=[tiempo de inicio],[tiempo de fin]. Admite el formato en horas:minutos:segundos o en segundos (como float).

  • "controls": Muestra los controles.Valor booleano (sin valor).
  • "loop": Empieza de nuevo al acabar.Valor booleano (sin valor).

  • "height": Alto (en px).
  • "poster": URL donde se ubica la imagen que se mostrará mientras se carga el video.

Etiqueta source

Especifica un recurso de audio o video. Se anida dentro de éstas.

Atributos

  • "src": La URL del recurso.
  • "type": El tipo de recurso utilizado.

Etiqueta track

Especifica una pista de texto para video. Sirve para poner subtitulos (por ejemplo). Se anida dentro de video.

Atributos

  • "default": la pista se habilitará si las preferencias del usuario no indican que otra pista sería más apropiada. Valor booleano (sin valor).
  • "src": La URL del recurso.
  • "label": Titulo de la pista de texto.
  • "srclang": Especifica el lenguaje de la pista de texto. Obligatorio si kind="subtitulos".
  • "kind": Especifica el tipo de pista de texto. Sus "valores" pueden ser:
    • "captions":Traducción para sordos (dialogo y efectos de sonido).
    • "chapters": Define titulos de capítulos (para navegación).
    • "descriptions": Descripción textual del contenido del video (para ciegos).
    • "metadata": Contenido en scripts. No es visible para el usuario.
    • "subtitles": el contenido es para poner subtitulos.

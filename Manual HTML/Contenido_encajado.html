Imágenes

Se utiliza la etiqueta img. .Únicamente se declara la etiqueta de apertura, no la de cierre (es una etiqueta vacía). Se puede anidar dentro de la etiqueta figure para sacarla del flujo HTM y añadir un comentario al pie con la etiqueta figcaption.

Atributos

  • src: Este atributo es obligatorio.En su "valor" se ha de especificar la URL de la imagen.
  • width: Ancho de la imagen. Su "valor" se especifica en píxeles (px).
  • heigth: Alto de la imagen. Su "valor" se especifica en píxeles (px).
  • alt: Texto alternativo que se muestra si la imagen no puede cargarse.
  • usemap: Sirve para definir la imagen como la utilizada para crear un mapa de imágenes. Su valor ha de ser # seguido del name del map.

Ejemplo de declaración

        <figure>
            <img src=gifs/hadoken.gif></img>
            <figcaption>¡HADOKEN!</figcaption>
        </figure>

Mapas de imágenes

Es una imágen, dividida en áreas en las que se puede hacer click. Se declara con la etiquetamap. Dentro de ella se anidan las etiquetas area que seccionaran la imagen en diferentes areas.

Atributos

El primer atributo, obligatorio, es name, que se coloca dentro de la etiqueta map. Es el nombre del mapa de imágenes. declararemos una imagen con el atributo usemap y en el valor de este pondremos el "#name_del_map"

El resto de los atributos se colocan dentro de las diferentes etiquetas area que se anidaran dentro de map. Únicamente se declara la etiqueta de apertura, no la de cierre (es una etiqueta vacía). Estos son:

  • "alt": Texto alternativo que se mostrará si no se muestra la imágen.Obligatorio si hay href.
  • "href": Hipervínculo del área. Es "obligatorio". O se pone este o nohref.
  • "nohref": Indica que no hay hipervínculo en el área. Es "obligatorio". O se pone este o href.
  • "shape": Forma del área. Los "valores" pueden ser:
    • "circle": Círculo
    • "poly": Polígono
    • "rect": Un rectangulo. No es soportada por todos los navegadores.
    • "default": Toda región mas allá de un área definida.
  • "coords": conjunto de valores que especifican el tamaño del área. Forma del área. Los valores van en píxeles (px). Segun la forma se declaran de una manera u otra:
    • Rectangulo: x-izquierda, y-arriba, x-derecha, y-abajo
    • Círculo: x-centro, y-centro, radio
    • Polígono: x1, y1, x2, y2,...,xn, yn
  • download: Igual que en los Hiperenlaces
  • target: Igual que en los Hiperenlaces

Ejemplo de declaración

            <img src=imagenes/paisajesimetrico.jpg usemap="#mapaimagenes">
            <map id="mapaimagenes" name="mapaimagenes">
                <area shape="rect" coords="0,0,910,198" alt="Cielo" href="https://es.wikipedia.org/wiki/Cielo" target="_blank">
                <area shape="rect" coords="0,199,910,430" alt="Cielo" href="https://es.wikipedia.org/wiki/%C3%81rbol" target="_blank">
                <area shape="rect" coords="0,431,910,606" alt="Cielo" href="https://es.wikipedia.org/wiki/Suelo" target="_blank">
            </map>

Iframes

Sirven para insertar otro documento dentro del documento HTML (como otro HTML). Se declaran con la etiqueta iframe.

Se recomienda añadir un texto dentro de la etiqueta para que se muestre si el navegador no soporta Iframes.

Atributos

Únicamente enumero los atributos soportados por HTML 5. Estos son:

  • src: En su "valor" se especifica la URL del documento.
  • srcdoc: En su "valor" se especifica el contenido HTML (en formato HTML) para que el iframe lo muestre.
  • width: Ancho del iframe. Su "valor" se especifica en píxeles (px).
  • heigth: Alto del iframe. Su "valor" se especifica en píxeles (px).
  • alt: Texto alternativo que se muestra si la imagen no puede cargarse.
  • sandbox: Impone restricciones adicionales al Iframe. Dentro de este atributo, se le pueden dar valores (booleanos), para que permita algunas de las acciones que ha restringido. Algunos valores son:
    • "": Sin valor. Aplica todas las restricciones al Iframe.
    • "allow-same-origin": Permite que el contenido del iframe sea tratado como del mismo origen.
    • "allow-top-navigation": Permite que el recurso navegue por un contexto de exploración de un nivel superior.
    • "allow-forms": Permite formularios.
    • "allow-scripts": Permite que se ejecuten scripts.
    • Lista completa de valores sandbox

Ejemplo de declaración

            <iframe srcdoc="<p>Esto es un Iframe</p>"
                Iframes no soportados.
            </iframe>

Embed

La etiqueta embed sirve para insertar una aplicación externa o de contenido interactivo (plug-in).

Atributos

  • "width": Anchura del plug-in.
  • "height": Altura del plug-in.
  • "src": URL externa donde esta ubicado el plug-in
  • "type": Tipo de contenido, es un valor media_typeLista de media_types.
  • "atributos propios del plug-in": Los valores variaran segun las especificaciones del plug-in.

Ejemplo de declaración

            <embed src= "flash/d12.swf>

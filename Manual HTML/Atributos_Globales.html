ATRIBUTOS GLOBALES FRECUENTES

En esta seccion definiremos algunos de los atributos globales más frecuentes que cualquier elemento HTML puede incorporar.

Para una lista completa de los atributos: TODOS LOS ATRIBUTOS HTML

TABLA RESUMEN

Atributo Utilización
class Nombre de una clase (o clases). se utiliza en conjunto a hojas de estilo y asigna una clase al elemento.
id Identificador único para un elemento, no puede haber otro igual en todo el documento HTML. Se utiliza para referirse al elemento desde vinculos, hojas de estilo...
lang El lenguaje utilizado en el contenido del elemento. Su valor ha de ser un lenguaje válido ("en", "es"...).
style Se utiliza para especificar el estilo de un elemento, como el color, tipo de letra, tamaño... sin necesidad de una clase, ni una hoja, de estilos. Se utiliza para aplicar estilos "in situ"
title Linea de texto que presenta información adicional sobre el elemento. Normalmente los navegadores lo muestran como una cajita emergente.
translate Un valor booleano ("yes" o "no"), que indica si el contenido del elemento debe ser traducido cuando el docuemnto es adaptado a la lengua local.
dir Un atributo que indica la direccionalidada del texto. Puede tener como valores:
  • "ltr": Para idiomas que se escriben de izquierda a derecha.
  • "rtl": Para idiomas que se escriben de deracha a izquierda.
  • "auto": Permite al navegador que decida segun el contenido de la etiqueta.
hidden Sirve para ocultar el elemento. Este elemento no se mostrará al usuario de la página. Es un valor "booleano", esta o no esta. Se declara sin valor (esto es, sin poner = detras de él) y no se hereda.

Declaración

Todos los atributos globales se declaran igual (con excepciones), dentro de la etiqueta de apertura de un elemento. Despues se coloca el atributo seguido de = y luego se declara un valor válido entre comillas dobles. Cuando existan varios atriburos, se concatenaran con espacios en blanco.

etiqueta atributo="valor del atributo"etiqueta

Selectores CSS

Referencia en HTML

Podemos referenciarnos a una etiqueta HTML de diferentes maneras:

  • A traves de un atributo id
  • A traves de un atributo class
  • A traves de una etiqueta nombre de la etiqueta

Declaracion de selectores en CSS

  • #id
  • .valor_de_una_clase (class).
  • * (todos los elementos HTML)
  • Enumeración: Si concatenamos etiquetas con , afectaremos a todas las enumeradas.
  • Una solo si dentro de otra: si las concatenamos sin comas sera solo la segunda que este dentro (anidada) en la primera.
